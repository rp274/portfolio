Quiz Web App

The goal of this project was to create a web app that was a functional ten question quiz. The user would be able to progress through the quiz and given feedback on correct and incorrect answers along the way. Following the last question the user has their final score displayed with the ability to retake the quiz.

Technologies used for this project include: HTML, CSS, JS, JQuery
